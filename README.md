# CustomItemsPlugin

Готовый плагин для Spigot/Paper с тремя кастомными предметами.

## Предметы

### 1. Прах убитого (WITHER_ROSE)
ПКМ — в радиусе 15 блоков всем игрокам даёт:
- Слепота — 5 сек
- Голод (Hunger) — 5 сек
- Отравление — 3 сек

Перезарядка: 60 сек.

### 2. Вспышка света (GLOWSTONE_DUST)
ПКМ — в радиусе 15 блоков всем игрокам даёт:
- Свечение — 30 сек
- Слабость — 3 сек

Перезарядка: 55 сек.

### 3. Иммунитет (SHIELD)
Пока предмет лежит где угодно в инвентаре игрока — отравление не действует.
Каждый раз, когда предмет "поглощает" яд, теряется прочность.
Когда прочность заканчивается — предмет ломается.

## Команда

```
/customitem <ash|flash|immunity> [ник]
```
Права: `customitems.give` (по умолчанию — только у операторов).

Примеры:
- `/customitem ash` — выдать себе Прах убитого
- `/customitem flash Steve` — выдать Вспышку света игроку Steve
- `/customitem immunity` — выдать себе Иммунитет

## Сборка

В этой песочнице нет доступа в интернет, поэтому собрать jar здесь нельзя —
Maven не сможет скачать `spigot-api`. Собери на своей машине:

1. Установи JDK 17+ и Maven.
2. Собери spigot-api через BuildTools (https://www.spigotmc.org/wiki/buildtools/)
   ИЛИ пропиши в `pom.xml` версию сервера, который уже стоит у тебя
   (например `1.20.4-R0.1-SNAPSHOT`, `1.21.1-R0.1-SNAPSHOT` и т.д.).
3. В папке проекта выполни:
   ```
   mvn clean package
   ```
4. Готовый файл появится в `target/CustomItemsPlugin.jar`.
5. Скопируй его в папку `plugins` твоего сервера и перезапусти сервер.

## Возможные доработки
- Исключить самого активировавшего игрока из зоны действия Праха/Вспышки.
- Добавить крафт для предметов вместо выдачи командой.
- Настроить радиус/время/кулдауны через config.yml.
